J&B Budget

Real-time sync setup guide — takes about 10 minutes total

STEP 1 — Supabase account (free)

Create your database

  • Go to supabase.com and click Start for free
  • Sign up and create a new project — name it jb-budget
  • Pick any password (save it) and choose the region closest to you
  • Wait ~1 minute for it to set up
STEP 2 — Create the tables

Run this SQL in Supabase

In your project, click SQL Editor in the left sidebar. Paste the code below and click Run:

create table bills (
  id uuid default gen_random_uuid() primary key,
  name text, amount numeric,
  type text, created_at timestamptz default now()
);
create table expenses (
  id uuid default gen_random_uuid() primary key,
  name text, amount numeric, paid text,
  type text, created_at timestamptz default now()
);
create table incomes (
  id uuid default gen_random_uuid() primary key,
  source text, amount numeric,
  who text, created_at timestamptz default now()
);
alter table bills enable row level security;
alter table expenses enable row level security;
alter table incomes enable row level security;
create policy "open" on bills for all using (true) with check (true);
create policy "open" on expenses for all using (true) with check (true);
create policy "open" on incomes for all using (true) with check (true);

You should see "Success. No rows returned." — that's correct!

STEP 3 — Get your keys

Copy your Project URL and API key

  • In Supabase, click Settings (gear icon) → API
  • Copy the Project URL (looks like https://abc123.supabase.co)
  • Copy the anon public key (long string starting with eyJ…)
STEP 4 — Host the app on GitHub Pages

Put the app online for free

  • Go to github.com → create a free account
  • Click + → New repository → name it jb-budget → Public → Create
  • Click uploading an existing file
  • Drag in all files from this zip (index.html, manifest.json, sw.js, icons folder)
  • Click Commit changes
  • Go to Settings → Pages → select branch main → Save
  • Your app will be live at https://YOUR-USERNAME.github.io/jb-budget
STEP 5 — Connect & install on both phones

Final step!

  • Open the GitHub Pages link on your phone
  • The app will ask for your Supabase URL and key — paste them in and tap Connect
  • It saves the keys on the phone — you only do this once per phone
  • Send the same link to B, she opens it, pastes the same keys, done!
  • Now both phones: tap Share → Add to Home Screen (Safari on iPhone)
✓ After setup, any expense Jacob adds instantly appears on B's phone and vice versa — live!
